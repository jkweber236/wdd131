Overview

Purpose

The purpose of this site will be to help users track their job applications, including their status and priority level.It will help users stay organized with their applications and make the job search process more manageable.

Audience

It is intended for job seekers who need an organized way to manage application details, track their status, and categorize applications based on their level of interest.

Dynamic elements

To add dynamic elements to the site, I will use Javascript to handle form submission for adding, updating, and removing job applications, with the data being stored for the user. Users will also be able to filter, search for, or sort job applications based on a couple factors such as date, priority level, name, company, etc.

Content

Home page

Header with logo, application name, and navigation links.

Form: A form for users to enter job application details with the following details:

  • Company Name
  • Status (Accepted, Pending, Rejected)
  • Interest Level (High, Medium, Low)
  • Application Date
  • Application Link
  • Additional Details

My Applications

Header with logo, application name, and navigation links.

List of all job applications that have been saved.

Displays interest level, company name and status. Also includes a drop-down for additional details (date, link, notes).

Buttons for delete and update will be displayed alongside each application entry. Form fields will be displayed once the update button is clicked to make changes to the existing content.

Wireframes

Home

The home page will merely display a form for the user to fill in application details to be stored and loaded in on the "my applications" page. The interest section will be a drop-down with 3 categories (high, medium, and low interest). The status will also be a drop-down with 3 categories (accepted, pending, and rejected).

My Applications

The "my applications" page will display details pertaining to each application that has been added. Initially, it will only show the interest, company name, and status. Each application will have a drop-down to show the rest of the related information such as application date, link, and additional details. Users will also have the option to update application details, including status changes, or delete an entry if needed.
